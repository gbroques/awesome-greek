## Greek Vowels

7 vowels
Α Ε Η Ι Ο Υ Ω

5 sounds
Α ah huzz**ah**
Ε ay hoor**ay**
ΗΙΥ ee fr**ee**
ΟΩ **o**
ΟΥ ooh sh**oe**

double vowels
ΑΙ ay hoor**ay**
ΟΙ ΕΙ ΥΙ ee fr**ee**
ΟΥ ooh sh**oe**
